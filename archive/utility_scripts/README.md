# Utility Scripts Archive

**Date:** January 2025  
**Purpose:** Archive utility scripts that are not part of the active production codebase

## Overview

This directory contains utility scripts that were created for debugging, monitoring, or one-off operations. These scripts are **not part of the active production codebase**.

## Archived Files

- `check_frontend_status.sh` - Utility to check frontend status
- `check_logs_after_test.sh` - Utility to check logs after tests
- `monitor_e2e_test.sh` - Utility to monitor E2E tests
- `restart_all_containers.sh` - Utility to restart containers
- `run_cto_demo_tests.sh` - Utility to run CTO demo tests
- `verify_containers.sh` - Utility to verify containers

## Active Scripts

Active production scripts are located in:
- `symphainy-platform/scripts/active/` - Production-ready scripts
- `symphainy-platform/scripts/evaluate/` - Scripts under evaluation

## Note

These scripts are kept for historical reference but should not be used for production operations. Use the active scripts in `symphainy-platform/scripts/` instead.
