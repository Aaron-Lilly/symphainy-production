# Platform Telemetry Terminology

**Date:** December 20, 2024  
**Purpose:** Clarify terminology for platform telemetry reporting vs. telemetry as data source

---

## ğŸ¯ Key Distinction

### **Platform Telemetry Events (Reporting)**
- **What:** Telemetry events **generated BY the platform** (errors, operations, metrics)
- **Purpose:** Report platform health, performance, and operational status to Nurse Service
- **Methods:**
  - `record_platform_error_event()` - Platform error events
  - `record_platform_operation_event()` - Platform operation events
  - `record_metric()` - Platform metrics
  - `log_health_metrics()` - Platform health metrics
  - `log_anomaly()` - Platform anomalies

### **Telemetry as Data Source (Ingestion)**
- **What:** Telemetry events **FROM external sources** (e.g., autonomous vehicles, IoT devices)
- **Purpose:** Ingested as data for analysis, processing, and business intelligence
- **Methods:** Data ingestion services (e.g., Data Steward, File Parser, etc.)

## ğŸ”§ Method Naming Convention

### **Platform Telemetry Reporting Methods**

**Error Events:**
- `record_platform_error_event(error_type, metadata)` âœ…
- Reports platform errors to Nurse for health monitoring

**Operation Events:**
- `record_platform_operation_event(operation_name, metadata)` âœ…
- Reports platform operations to Nurse for metrics tracking

**Metrics:**
- `record_metric(metric_name, value, tags)` âœ…
- Reports platform metrics to Nurse

**Health:**
- `log_health_metrics(metrics)` âœ…
- Reports platform health status to Nurse

**Anomalies:**
- `log_anomaly(anomaly_data)` âœ…
- Reports platform anomalies to Nurse

### **Deprecated Methods (Backward Compatibility)**

- `record_telemetry_event(event_name, data)` âš ï¸ **DEPRECATED**
  - Still works but delegates to platform-specific methods
  - Services should migrate to `record_platform_error_event()` or `record_platform_operation_event()`

## ğŸ“ Usage Examples

### **Platform Error Event (Recommended)**
```python
# Report platform error to Nurse
await self.telemetry.record_platform_error_event("error_occurred", {
    "operation": "parse_file",
    "error_type": "ValidationError",
    "error_message": "Invalid file format"
})
```

### **Platform Operation Event (Recommended)**
```python
# Report platform operation to Nurse
await self.telemetry.record_platform_operation_event("file_parsed", {
    "file_id": "file_123",
    "file_type": "pdf",
    "duration": 1.5
})
```

### **Deprecated (Still Works)**
```python
# Still works but deprecated
await self.record_telemetry_event("file_parsed", {...})
# Delegates to record_platform_operation_event()
```
