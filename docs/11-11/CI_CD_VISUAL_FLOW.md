## ğŸ“ **KEY CONCEPTS**

### **Phase Gate:**
A checkpoint that code must pass through. Like airport security.

### **Artifact:**
Something saved from the pipeline (screenshots, videos, reports).

### **Blue-Green Deployment:**
Running two versions side-by-side, switching traffic only when new version is healthy.

### **Smoke Test:**
Quick validation that basic functionality works. Like checking "does it turn on?"

### **Rollback:**
Reverting to the previous version if something goes wrong.

### **Continuous:**
Happening automatically and frequently (not manual and monthly).
